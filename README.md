一个基于 Go (Gin) 实现的简易 HTTP 负载均衡器。
它支持动态健康检查、轮询算法（Round-Robin）以及故障自动剔除。

##  功能特性

* ** 高性能转发**：基于 `net/http/httputil` 实现高效的反向代理。
* ** 轮询算法**：使用原子计数器（Atomic）实现线程安全的 Round-Robin 负载均衡。
* ** 主动健康检查**：后台协程每 2 秒探测一次后端服务器 TCP 连接，实时感知节点状态。
* ** 故障剔除与恢复**：
    * 自动剔除“挂掉”的节点。
    * 节点恢复后自动重新加入负载池。
    * 全挂时返回友好的 502 错误页面。
* ** 配置文件驱动**：通过 `config.json` 灵活管理端口和后端地址。
